Introduction

Based on TEX Latex is a document preparation system widely used in academia specially for writing mathematical and non latin symbols. Latex was initially developed by Leslie Lamport in 1984 and is being used worldwide for several decades.
Latex has a simple syntax, supports thousands of symbols and is one of basic and important skill.

Front Page

Now lets make front page for our document like we do in word processors. First define:

  • 1 - title using '\title{myTitle}'.
  • 2 - date using '\date{2015-04-08}.
  • 3 - author using '\author{author name}'.

Now you can insert title in document using '\maketitle'. To insert new page use '\newpage'.

Document Structure

For structuring the document we have:

  • 1 - Section, Latex code" \section{Sample Section}.
  • 2 - Sub-Section, Latex code" \subsection{Sample Subsection}.
  • 3 - Sub-Sub-Section, Latex code" \subsubsection{Sample Subsubsection}.
  • 4 - Paragraph, Latex code" \paragraph{Sample Paragraph}.
  • 5 - Sub-Paragraph, Latex code" \subparagraph{Sample Subparagraph}.

Text inside curly brackets is displayed as heading for structure, for no heading use empty brackets. For example \paragraph{} will create paragraph without any heading.

Equations

Inserting mathematical equations in document is simple, just write equations between '\begin{equation} and \end{equation}' and you will get nicely rendered equations along with equation numbers.

Next

Packages

Another great thing about Latex is that its functionality can be extended using packages. To use a package in your document you need to write 'usepackage{packagename}' before '\begin{document}'. For example if you want to add equations without any numbering you can use 'equation*' provided by 'amsmath' package. (Note that there is no equation number in output).

Next

Adding Images

Adding images in Latex is simple, just use 'graphicx' package and insert image using 'figure'. 'figure' has some parameters such as \includegraphics{locationofimage}, caption{captiontext} and label{labeltext}.
